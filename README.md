# 📌 Project Overview

This project analyzes real-world COVID-19 data to understand the spread of the pandemic across different countries. Using Python libraries such as Pandas and Matplotlib, the data was cleaned, filtered, analyzed, and visualized to identify trends in confirmed cases and deaths.

The analysis focuses on comparing the COVID-19 situation in **India**, **United States**, and **Brazil** during **2020**, helping demonstrate practical data analysis and visualization techniques.

---

# 🎯 Objectives

- Analyze COVID-19 case trends
- Compare infection rates between countries
- Study daily new cases
- Calculate rolling averages to smooth fluctuations
- Visualize pandemic growth using line charts
- Generate meaningful business/public health insights

# 📊 Key Insights

### 🇺🇸 United States

- Recorded the highest total COVID-19 cases during 2020.
- Experienced multiple large infection waves.

---

### 🇮🇳 India

- Showed rapid case growth during the second half of 2020.
- Daily cases increased significantly before gradually declining.

---

### 🇧🇷 Brazil

- Maintained consistently high infection rates throughout the year.
- Followed a growth pattern similar to India during peak periods.

---

### 📉 Daily Cases

- Daily new cases fluctuated considerably.
- The 7-day rolling average helped smooth short-term variations and highlighted long-term trends.

---

### 📈 Overall Trend

All three countries experienced exponential growth in COVID-19 cases during 2020, demonstrating the importance of public health measures and continuous monitoring.
